1.Направете база през phpMyAdmin след това импортнете базата books_comments.sql

2.Конфигурацията за базата е в includes/connection.php
ако вашата се различава от:

'DB_HOST' => 'localhost',
'DB_USER' => 'root',
'DB_PASSWORD' => '',
'DB_NAME' => 'books_comments'

нанесете вашите данни там, за да сработи базата с проекта.
